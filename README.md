# Issa

A fresh [Next.js](https://nextjs.org) App Router workspace for shaping and building the product.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `npm run dev` starts the local development server.
- `npm run lint` runs ESLint.
- `npm run build` creates a production build with webpack for a more stable local CI path.
